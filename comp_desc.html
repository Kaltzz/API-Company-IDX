Companies Description

Companies Description digunakan untuk mendapatkan informasi spesifik mengenai satu perusahaan berdasarkan Industri Pengguna dapat menggunakan Industri sebagai parameter pada URL untuk mengakses data perusahaan tertentu berdasarkan industri perusahaan tersebut.

Request

Parameter Type Position # Description
key Integer Header Required API key
Industri Varchar Params Not Required Industri Perusahaan
