Search Companies

Search Companies memungkinkan pengguna mencari perusahaan berdasarkan kriteria tertentu, seperti nama perusahaan, sektor industri, atau kode saham. Pengguna dapat memberikan parameter pencarian pada URL untuk memfilter hasil pencarian.

Request

Parameter Type Position # Description
Key Integer Header Required API key
account status Integer Body Not Required ID user account

Response

{
    "status": true,
    "result_code": 200,
    "message": "Success!",
    "values": {
        "name": "budi",
        "email": "budi@gmail.com",
        "token": "9WUzKE7kCI1vSuQAbrmOwc2m2dk1NbPR",
        "account_status": "1"
    }
}
