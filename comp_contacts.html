Companies Contacts

Companies Contacts digunakan untuk mendapatkan informasi spesifik mengenai kontak satu perusahaan seperti no telpon, email dan lain-lain berdasarkan alamat Pengguna dapat menggunakan alamat sebagai parameter pada URL untuk mengakses data kontak perusahaan yang dicari.

Request

Parameter Type Position # Description
key Integer Header Required API key
alamat Varchar Params Not Required Alamat perusahaan
